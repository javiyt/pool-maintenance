# pool-maintenance

A basic Express.js backend server for pool maintenance applications.

## Getting Started

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm start
```

Access endpoints at: http://localhost:3000/health

## API Endpoints

- `/` - Welcome message with application info
- `/health` - Health check status
- `/static/*` - Serve static frontend files from public/ directory
